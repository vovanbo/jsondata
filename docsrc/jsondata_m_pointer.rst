Operators
---------

The syntax displayed for provided operators is::

  S: self
  x: parameter
  n: numerical parameter for shift operators.

Thus the position of the opreator and parameteres is defined as follows::

   z = S + x: LHS: __add__
   z = x + S: RHS: __radd__
   S += x:    LHS: __iadd__
